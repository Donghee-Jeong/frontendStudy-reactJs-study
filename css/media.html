700px보다 작을때는 빨강, 클때는 파랑
        @media 미디어타입 and (미디어특성){
            css코드
        }

        미디어타입
        1. all : 모든 미디어 타입(기본값)
        2. screen : 컴퓨터화면,타블렛,스마트폰 등
        3. print : 인쇄 전용

        미디어특성
        1. width[height] : 뷰포트 가로[세로] 너비
        2. max-width[height] : 뷰포트 최대 가로[세로] 너비(이하)
        3. min-width[height] : 뷰포트 최소 가로[세로] 너비(이상)
        4. orientation : 뷰포트 방향(portrait, landscape)
